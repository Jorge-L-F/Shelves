Place of Gathering - Zoom: https://videoconf-colibri.zoom.us/j/55465464522

Description
In this event one person will read out aloud the first chapter and we will discuss our thoughts and opinions on what was read. Afterwards someone else will read the second chapter and so on. The goal of this meeting is to discuss different opinions about the book.

Warnings/Rules for the Event
While someone is reading there should be no interruptions. All other microphone should be on mute. Questions should be posted on the Zoom chat and will be answered in order. If you want to talk during the discussion use the "raise hand" symbol and wait to be called, this also be done in order of who asked first in order to avoid confusion.
